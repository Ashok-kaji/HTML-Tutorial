*  INTERNET - Interconnected network of computers around the world.

*  In wi-fi what happens is - all computers around the world are Interconnected by wires or cables, and form
   a network of computers, which allow them to communicate. wi-fi companies or INTERNET providers use cables
   upto our home to bring INTERNET and let us connect with other computers, so that we can communicate with 
   other computers.

*   Hamare ghar par INTERNET ko lane ke liye hm cables ka use karte h. Isliye jab v wi-fi, router set-up hota h,
    ya hme apne devices par INTERNET use karna hota h, to hmehsa bahar ki companies se ek cable aata h, jise ghar 
    me joda jata hai. fir us cable se hm apne devices ko by wire or wirelessly connect karte hai , aur fir 
    hm baki computers ke sath connect ho jate h, aur INTERNET use karne lagte h, yani baki computers ke sath 
    communicate karne lagte hai.

*   Server - a device or system whic is always online and serve other computers through INTERNET.

*   Client - any device which ask information from server.

*   IP Address - each & every computers connected to INTERNET having a unique Address called IP Address. And, due to this 
    IP Address we know with which computer we are communicating. and all computers are trackable using IP Address
    over INTERNET.

*   IP Address is simply a Address of any Websites.

************** What is WEB DEVELOPMENT ?? ****************

----> Building websites for the INTERNET.

----> Request - Response Cycle :- Hm jis v website ko apne browser me search karte hai......
                                1. To us website ke server ko hm ek request send karte h.
                                2. Fir wo Server hme ek Response send karta h, aur Response me server website nhi bhejta h, wo hme website
                                   ki codes ka bundle(Html,css,JS) send karta h.
                                3. Hme Response send karne ka mtlb ki hmare browser ko Response send karega aur fir hmara browser 
                                   us code ke bundle ko ek pura website me convert karega hmare computer me,
                                    aur fir hme apne browser me wo pura website dikhega.

----> View Page Source:- we can see the Codes which server has sent to our browser, and browser converted that codes into a website.

----> 1. HTML(structure)           2. CSS(styling)         3. JavaScript(Functionality)

----> Frontend technologies:- HTML, CSS, JavaScript
----> Frontend:- In request-Response cycle, client side(browser-request, code rendering-form change karke display karana.) is known as Frontend.
----> To Frontend wo wala hissa hota h hmari website ka jiske sath hm interact kar pate h,Jo hme hmare browser ke uper dikhayi deta h, 
      jiske under html,css and JavaScript hoti h.
----> Backend:- In request-Response cycle, server side is known as Backend.
